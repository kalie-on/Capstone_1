# Capstone_1_Analysing_EmporiUm_Sale_Data
# West region, Colorado Territory
# Manager Jim Heck Student Ndifon Caleb

This project focuses on analyzing sales data for EmporiUm, a student bookstore chain that offers a wide range of products including books, technology, and a supplies through in-store and online channels. The purpose of the analysis is to better understand sales performance across different regions and sales territories, helping the business make better decisions based on data.

In this project, i will be using the following tools;
MY SQL,
Excel,
GitHub and
Git Bash.

Using SQL, sales data will be queried to answer quetions like total revenue by store, rankings of in-store and online sales performance, and revenue generated by product categories. Results will be exported into CSV format and used in Excel to create Pivot Charts, providing a visual representation of trends and patterns in the data, Git Bash will be used to push the work from local to remote repository; GitHub.

This analysis will highlight top-performing stores, states, and product categories, giving guidance for improving sales strategies, inventory management, and overall business progress. This project demonstrates the use of data analysis and visualization tools to turn raw data into meaningful business visuals.
